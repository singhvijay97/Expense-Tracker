# Expense-Tracker
# Objective:
Build a small Python program that allows users to add daily expenses, view them, compute totals, and export to a CSV file.

# FEATURES

1.Add expense (amount, category, note)
2.View all expenses
3.Show total spent
4.Export data to CSV file
5.Exit

# REQUIRED MODULES

1.csv (standard library)
2.datetime (standard library)
